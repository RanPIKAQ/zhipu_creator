# zhipu_creator

## 通过 docker 部署

1. `git clone https://github.com/RanPIKAQ/zhipu_creator.git`
2. `cd zhipu_creator`
3. `sudo docker compose up -d`


## 本地部署

1. `git clone https://github.com/RanPIKAQ/zhipu_creator.git`
2. `cd zhipu_creator`
3. `pip install -r requirements.txt`
4. `python app.py`
5. 访问 `127.0.0.1:5000`
